Bootstrap 按钮样式

可用于<a>, <button>, 或 <input> 元素上
.btn 为按钮添加基本样式
.btn-default 默认/标准按钮
.btn-primary 原始按钮样式（未被操作）
.btn-success 表示成功的动作
.btn-info 该样式可用于要弹出信息的按钮
.btn-warning 表示需要谨慎操作的按钮
.btn-danger 表示一个危险动作的按钮操作
.btn-link 让按钮看起来像个链接
.btn-lg 制作一个大按钮
.btn-sm 制作一个小按钮
.btn-xs 制作一个超小按钮
.btn-block 块级按钮(拉伸至父元素100%的宽度)
.active 按钮被点击
.disabled 禁用按钮

Bootstrap 表单样式

  • <form> 元素添加 role="form"
  • 把标签和控件放在 <div class="form-group" > </div>中
  • 文本元素 <input>、<textarea> 和 <select> 添加 class="form-control"
  • input 添加 disabled 属性，会改变输入框的样式以及当鼠标悬停的样式
  • 使用<fieldset disabled> <fieldset>来禁用标签内的所有控件
  • 对父元素简单地添加适当的 class（.has-warning、 .has-error 或 .has-success）即可使用验证状态
  • 可使用 class .input-lg 和 .col-lg-* 来设置表单的高度和宽度
  • 可在 input 后使用 .help-block 加一个块级帮助文本

Bootstrap 表格样式

表格元素
<table> 为表格添加基础样式。
<thead> 表格标题行的容器元素（<tr>），标识表格列
<tbody> 表格主体中的表格行的容器元素（<tr>）
<tr> 一组出现在单行上的表格单元格的容器元素
<td> 默认的表格单元格
<th> 特殊单元格，标识列或行。必须在<thead>使用
<caption> 关于表格存储内容的描述或总结
表格样式
.table-responsive 把任意的 .table 包在其中，便可响应式
.table 为任意 <table> 添加基本样式 (只有横向分隔线)
.table-striped 在 <tbody> 内添加斑马线形式的条纹 ( IE8 不支持)
.table-bordered 为所有表格的单元格添加边框
.table-hover 在 <tbody> 内的任一行启用鼠标悬停状态
.table-condensed 让表格更加紧凑
<tr>, <th> 和 <td>样式
.active 将悬停的颜色应用在行或者单元格上
.success 表示成功的操作
.info 表示信息变化的操作
.danger 表示一个警告的操作
.warning 表示一个危险的操作

Bootstrap 图片样式

  • 通过在 标签添加 .img-responsive 类来让图片支持响应式设计。 图片将很好地扩展到父元素
.img-rounded 为图片添加圆角 (IE8 不支持)
.img-circle 将图片变为圆形 (IE8 不支持)
.img-thumbnail 缩略图功能
.img-responsive 图片响应式 (将很好地扩展到父元素)

Bootstrap 排版样式

  • 定义（h1 到 h6）样式，若在内部加<small>，或加class="small"就能得到小一个字号的颜色更浅的文本
  • 强调标签 <strong>（设为更粗的文本）、<em>（设为斜体）
  • <abbr> 在文本底部的一条虚线边框。加 class="initialism" 可悬浮更小字体的title
  • <address> 标签显示联系信息
  • <blockquote> 及一个 <small> 标签来标识引用的来源
.lead 使段落突出显示
.small 设定小文本 (设置为父文本的 85% 大小)
.text-left 设定文本左对齐
.text-center 设定文本居中对齐
.text-right 设定文本右对齐
.text-justify 设定文本对齐,段落中超出屏幕部分文字自动换行
.text-nowrap 段落中超出屏幕部分不换行
.text-lowercase 设定文本小写
.text-uppercase 设定文本大写
.text-centerapitalize 设定单词首字母大写
.initialism 显示在 <abbr> 元素中的文本以小号字体展示
.blockquote-reverse 设定引用右对齐
.list-unstyled 移除默认的列表样式，列表项中左对齐 ( <ul> 和 <ol> 中)
.list-inline 将所有列表项放置同一行
.dl-horizontal 该类设置了浮动和偏移，应用于 <dl> 元素和 <dt> 元素
.pre-scrollable 使 <pre> 元素可滚动 scrollable

Bootstrap 网格系统--布局样式

基本的网格结构

下面是 Bootstrap 网格的基本结构：

<div class="container">
    <div class="row">
    <div class="col-*-*"></div>
        <div class="col-*-*"></div>      
    </div>
    <div class="row">...</div>
</div>
<div class="container">
    ....
</div>
  • 【偏移列】使用 .col-md-offset-* 类 进行偏移设置
  • 【嵌套列】在已有的 .col-md-* 列内添加新的 .row，与一组 .col-md-* 列
  • 【列排序（难点）】改变带有 .col-md-push-* 和 .col-md-pull-* 类的内置网格列的顺序

18、导航栏

  • 设置标签为 <nav class="navbar navbar-default" role="navigation">
  • navbar-fixed-top：固定顶部；navbar-fixed-bottom：固定底部
  • navbar-static-top：静态顶部，能随着页面一起滚动的导航栏
  • navbar-inverse：导航颜色倒置，黑底白字
  • <div class="navbar-header" > 内包含带有 <a class="navbar-brand">这让文本更大一号
  • 添加< ul class="nav navbar-nav" > </ul> < li class="active">表示选中，disabled 表示禁用
  • navbar-left 或 navbar-right 左浮动/右浮动
  • 响应式
  • 折叠的内容必须在 <div class="collapse navbar-collapse" > </div>中
  • 折叠按钮： class="navbar-toggle" data-toggle="collapse" data-target="#ID值"
  • 折叠按钮包括 3个<span class="icon-bar"></span>组成汉堡按钮
  • 导航栏中的表单
  • <form class="navbar-form navbar-left" role="search"> </form>
  • 导航栏中的按钮
  • 按钮在导航栏上垂直居中 class="navbar-btn" 可用在 <a> 和 <input> 上
  • 导航栏中的文本
  • class="navbar-text"，通常与 <p> 标签一起使用
  • 非导航链接
  • class="navbar-link"，改变<a>样式
